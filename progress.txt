Backend work completed (backend/)
- Initialized Node project, installed express + cors + nodemon
- Implemented routes:
  - GET /health -> { ok: true }
  - GET /search?q=query
    - Uses yt-dlp with ytsearch to return JSON list:
      - title, videoId, thumbnail, duration, channel
  - GET /stream?videoId=xxx&quality=low|high
    - Uses yt-dlp --get-url and format selection:
      - high: bestaudio
      - low: worstaudio (tries to prefer <=64kbps)
    - Returns { url } (note: stream URLs expire)
- CORS configured for frontend dev origin (default http://localhost:5173)
- Files:
  - backend/index.js
  - backend/package.json scripts: dev/start

Prerequisites / important notes
- yt-dlp is REQUIRED for backend search/stream and was NOT installed on this machine when checked.
  - Install:
    - pipx install yt-dlp
    - or python3 -m pip install -U yt-dlp
- Stream URLs can expire; if playback stops later, pressing Play again should re-fetch.
